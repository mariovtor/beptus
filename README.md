# beptus.com

This repository is the source for **[beptus.com](https://beptus.com)** — a GitHub Pages site used exclusively to host app metadata required by app stores and ad networks.

## What's here

| File / Path | Purpose |
|---|---|
| `worshippad/privacy.html` | Privacy policy for the Worship Pad app |
| `worshippad/support.html` | Support page for the Worship Pad app |
| `worshippad/terms.html` | Terms of use for the Worship Pad app |
| `app-ads.txt` | Authorized ad sellers declaration (Google) |
| `CNAME` | Custom domain configuration |

The `index.html` is a temporary placeholder. A proper site is being built separately.

## Not the main codebase

App source code lives in separate private repositories. This repo contains only the public-facing metadata pages that stores and platforms require to be reachable via a stable URL.
